# 📚 FlashCards

**FlashCards** — це вебзастосунок для вивчення термінів та іноземних слів. Від іноземних слів і граматичних конструкцій до математичних формул та хімічних рівнянь — користувачі можуть створювати власні набори карток з будь-яким вмістом і вивчати їх у зручній, інтерактивній формі.

Також доступні 4 режими вичення :
- Режим карток
- Письмовий режим
- Режим встановлення відповідностей
- Тестовий режим

Доступний функціонал для навчальних груп, який відкриває нові можливості для колективного навчання, взаємодії викладачів і
студентів. Викладачі створюють набори карток і завдання, а студенти навчаються.

## ✨ Основні можливості

- 📁 Створення та редагування власних наборів карток  
- 🧠 Режими інтерактивного навчання  
- 🎨 Інтуїтивний та доступний інтерфейс  
- 🔍 Пошук і фільтрація карток  


## 🛠 Технології

- **Frontend**: [React](https://reactjs.org/)
- **Backend**: [Flask](https://flask.palletsprojects.com/)
- **База даних**: [PostgreSQL](https://www.postgresql.org/)
